2016 Summer Open Source Software Workshop

주최: 한국정보과학회 오픈소스소프트웨어 연구회(http://sigoss.github.io)

일시: 2016년 8월 23일(화) ~ 8월 24일(수) 9:00 ~ 18:00

장소: 토즈 강남2호점

후원/협찬 :

오픈소스 소프트웨어 워크샵

  • 워크샵 개요 :

    오픈소스 소프트웨어 방식의 연구 개발을 수행하는 연구 책임자들과 개발자들이 오픈소스 소프트웨어 철학과 중요성, 오픈소스 커뮤니티, 오픈소스 소프트웨어 라이선스/거버넌스, 오픈소스 방식의 프로젝트 운영 프랙티스를 사례 중심으로 배우고 토론한다.
    개발에 직접 참여하는 연구자, 개발자들을 위한 커뮤니티 방식의 개발을 위한 도구와 개발 방법론을 완전 기초부터 실습으로 익힌다.

  • 강의 목표 :

    오픈소스 소프트웨어와 커뮤니티, 라이선스, 지적재산권 이슈를 이해한다.
    오픈소스 방식의 R&D 진행을 위한 활동 요소들을 파악하고 준비할 수 있다.
    커뮤니티 방식의 개발 프랙티스의 기본을 배워 적용할 수 있다.

  • 세부 일정 :

    1일차:

    • 9:00 오픈소스 소프트웨어 철학과 중요성 (1시간, 강사: 이민석)
    • 10:00 오픈소스 라이선스/특허/법적 쟁점 (2시간, 강사: 이철남)
    • 점심 식사 (1시간 30분)
    • 13:30 오픈소스 소프트웨어 개발 프랙티스 (1시간 30분, 강사: 박민우)
    • 15:00 오픈소스 소프트웨어 커뮤니티 활성화 사례/어려움 (1시간, 강사: 허광남)
    • 16:00 오픈소스 방식 프로젝트 운영, 평가 방향 (1시간 30분~, 발제:이민석, 토론 : 허광남, 참가자 전원)
    2일차:
    • SW 개발 프랙티스 튜토리얼 (8시간) (강사: 정인상, 조교들)
      • 오픈소스 소프트웨어 개발 라이프사이클 도구 소개 (1시간)
      • 간단한 예를 이용한 도구의 사용 실습
        • 소스 버전 관리 (git) : commit, branching/merge, reset, …
        • 이슈관리, 협업 (github) : clone, fork/pull, issue tracking, …
        • 지속적 통합 (Travis) : Continuous Integration, github 연동

  • 주 참가대상 :

    1일차 : 대학, 연구소의 과제 책임자, Senior/박사과정 연구원
    2일차 : 누구나 (교수/대학원생/학부생/연구원/초급개발자)

  • 참석자 제공 내역 :

    강의 자료 (인쇄본 및 편집 가능한 슬라이드 원본 파일)
    점심 식사 및 간식

  • 참가자 준비물 :

    1일차 : 없음
    2일차 : 실습을 위한 노트북 (무겁더라도 성능이 좋은 걸로, 하드 10GB 여유)
